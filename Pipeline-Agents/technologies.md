# Technology Stack

## Frontend (Framework, Styling)
- **Framework:** React
  - **Justification:** React is a widely-used, modern JavaScript library that allows for the creation of dynamic and responsive user interfaces. It is well-suited for building simple landing pages due to its component-based architecture, which promotes reusability and maintainability.
- **Styling:** Tailwind CSS
  - **Justification:** Tailwind CSS is a utility-first CSS framework that allows for rapid styling and customization. It is ideal for creating clean, modern designs and ensures responsiveness across different devices.

## Backend (Language, Framework)
- **Language:** Node.js
  - **Justification:** Node.js is a popular choice for server-side applications due to its non-blocking, event-driven architecture. It is lightweight and efficient, making it suitable for handling form submissions and sending email notifications.
- **Framework:** Express.js
  - **Justification:** Express.js is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications. It is perfect for handling simple form submissions and routing.

## Database
- **Database:** SQLite
  - **Justification:** SQLite is a lightweight, serverless database engine that is easy to set up and use. It is suitable for storing form submissions in a simple table without the overhead of a full-fledged database server.

## Other Tools
- **Email Service:** SendGrid
  - **Justification:** SendGrid is a reliable email delivery service that can be easily integrated with Node.js applications to send email notifications upon form submission.
- **Content Delivery Network (CDN):** Cloudflare
  - **Justification:** Cloudflare provides a robust CDN solution that can efficiently deliver static assets, ensuring fast load times and improved performance for the landing page.
